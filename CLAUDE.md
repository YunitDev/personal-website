## Brand Colors

```
Red:    #E85D5D  (primary, CTAs, Benson)
Blue:   #4A90D9  (secondary, YouTube)
Purple: #9B6BC4  (accent, Writings)
```

## Dark Theme (Default)

```
Background: #09090B    Text:       #FAFAFA
Surface:    #0F0F11    Secondary:  #A1A1AA
Card:       #18181B    Muted:      #71717A
Border:     #27272A
```

## Typography

- **Font:** Inter (weights 400, 500 only)
- **Never** use font-weight 600 for headings
- Section labels: uppercase, wide tracking, 12px

## Grid System

- 6-column grid, 10px gap
- Row height: 75px (content), 70px (tools)
- Container: max-width 672px, padding 20px

## Card Sizes

| Size | Class |
|------|-------|
| Hero | `col-span-6 row-span-2` |
| Large | `col-span-4 row-span-2` |
| Medium | `col-span-2 row-span-2` |
| Wide | `col-span-4 row-span-1` |
| Half | `col-span-3 row-span-1` |
| Small | `col-span-2 row-span-1` |

## Animations

- Duration: 0.8s
- Easing: `cubic-bezier(0.16, 1, 0.3, 1)`
- Stagger: +50ms per element
- Hover: `scale-[1.02]` for small cards, `scale-[1.005]` for large

## Octopus Marks

1. **The Reach** - Full 8-tentacle octopus (header, brand presence)
2. **The Spiral** - Coiled form with gradient (loading, growth)
3. **The Eye** - Watching eye (insights, analytics)

## Rules

- Colored cards always use white text
- Never use brand colors for body text
- Decorations: 10-20% opacity, increase on hover
- SVG animations: 2-4s duration, infinite repeat
- Profile image is grayscale
